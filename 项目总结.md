# MatAgent-MCP 项目总结

## 项目概述

MatAgent-MCP 是一个基于 LangChain 和 LangGraph 的材料科学智能设计与计算平台。项目集成了材料数据库查询、晶体结构建模、机器学习预测、第一性原理计算（VASP）任务管理等功能，提供了一个完整的材料研发工作流自动化解决方案。

**核心特色：**
- 🧠 基于 LangGraph 的 Function Calling Agent 架构
- 🔍 多数据库材料搜索（Materials Project、OQMD）
- 🏗️ 交互式晶体结构建模与可视化
- 🤖 机器学习模型（XGBoost）带隙预测
- 💻 VASP 远程计算任务全流程管理
- 🌐 Streamlit Web 应用界面
- 📁 集成文件服务器用于 2D/3D 结构可视化

## 核心技术架构

### 1. LangGraph Function Calling Agent (`langchain_agent.py`)

**架构设计：**
- 基于 LangGraph 显式图结构，使用 `StateGraph` 和 `MessagesState`
- 采用标准 ReAct（Reasoning + Acting）模式
- 包含 `assistant_node`（LLM调用）和 `tools_node`（工具执行）
- 支持上下文记忆（最近20条消息）

**核心组件：**
- `MatAgent` 类：主代理类，管理工具调用和状态图
- `AgentState`：扩展的 LangGraph 状态，包含消息历史
- `create_langchain_agent()`：工厂函数创建代理实例

**工具系统（28个工具函数）：**

| 类别 | 工具函数 | 功能描述 |
|------|----------|----------|
| **基础工具** | `get_time()` | 获取系统时间 |
| **材料搜索** | `search_materials_from_mp()` | Materials Project 材料搜索 |
| | `search_materials_from_oqmd()` | OQMD 数据库搜索 |
| | `get_material_project_page()` | 获取 Material Project 页面链接 |
| **结构获取** | `get_material_structure_from_mp()` | 获取 MP 材料结构（含2D/3D可视化） |
| | `get_material_structure_from_oqmd()` | 获取 OQMD 材料结构 |
| | `get_band_gap()` | 获取材料带隙值 |
| | `get_material_all_information_by_id()` | 获取材料完整信息 |
| **结构建模** | `build_structure()` | 根据晶格参数构建晶体结构 |
| | `get_structure_plot()` | 生成2D结构图（返回图片URL） |
| **ML预测** | `predict_band_gap()` | 使用 XGBoost 预测带隙 |
| **VASP任务** | `create_task()` | 创建远程VASP任务目录 |
| | `list_task_directories()` | 列出远程任务目录 |
| | `check_squeue()` | 检查 Slurm 任务队列 |
| | `execute_command()` | 在计算服务器执行命令 |
| | `extract_file()` | 从服务器提取文件 |
| | `create_mission()` | 创建计算任务输入文件 |
| | `submit_mission()` | 提交计算任务 |
| | `modify_incar()` | 读写/修改 INCAR 文件 |
| | `extract_result()` | 提取计算结果 |
| **项目管理** | `list_all_projects()` | 列出所有材料研发项目 |
| | `set_task_progress()` | 记录项目进度 |
| | `get_project_workflow()` | 查看项目详情 |
| **文件操作** | `read_file()` | 读取本地文件 |

**LLM 配置：**
- 使用 DeepSeek API (`deepseek-chat`)
- 绑定所有工具函数，支持 function calling
- 系统提示词包含可视化URL格式规则

### 2. Streamlit Web 应用 (`web_app.py`)

**界面设计：**
- 响应式布局，侧边栏功能导航
- 五个主要功能面板：
  1. **💬 AI对话** - 自然语言交互，实时显示工具调用
  2. **🔍 材料查询** - 多条件材料搜索与结构可视化
  3. **📊 结构建模** - 交互式晶体结构构建
  4. **🧪 ML预测** - 带隙预测与其他ML功能
  5. **💻 VASP任务** - 完整的VASP计算工作流管理

**核心功能模块：**

#### AI对话界面
- 实时流式响应
- 工具调用结果可视化展示（参数、结果、图表）
- 支持2D/3D结构图嵌入显示
- 会话历史管理与持久化存储

#### 材料查询面板
- 多条件组合搜索：元素、排除元素、化学系统、带隙范围
- 搜索结果表格展示
- 一键查看结构（晶格参数、空间群、原子位点）
- 2D结构图显示与3D结构交互链接

#### 结构建模面板
- 晶格参数输入（a,b,c,α,β,γ）
- 原子信息输入（元素符号、分数坐标）
- 超胞扩展因子设置
- 支持保存为CIF文件

#### VASP任务管理面板
- **6个标签页的完整工作流：**
  1. 📁 历史任务 - 浏览现有任务目录
  2. 📊 任务队列 - 查看Slurm队列状态
  3. ➕ 创建任务 - 新建任务目录并上传CIF
  4. 📝 生成输入 - 创建计算任务输入文件（relax/scf/band/dos）
  5. ⚙️ 修改INCAR - 交互式INCAR参数编辑（表格编辑器）
  6. 🚀 提交/提取 - 提交任务与提取计算结果

#### 数据持久化
- 使用SQLite数据库存储会话历史
- 支持多会话切换与管理
- 自动加载最近会话

### 3. 辅助模块

**Flask 文件服务器 (`flask_server.py`):**
- 提供静态文件服务，用于结构可视化
- 支持HTML（3D结构）、PNG（2D结构）文件上传
- 生成可访问的URL供Web应用使用

**SSH连接管理 (`tryssh.py`):**
- VASP任务远程连接与文件传输
- 支持Slurm作业提交与管理
- INCAR文件读写与参数修改
- 计算结果提取与解析

**环境配置 (`loadenv.py`):**
- 统一管理API密钥、SSH连接参数
- 支持环境变量和配置文件

**数据库管理 (`databasemanage.py`):**
- SQLite数据库操作
- 会话历史和聊天记录存储

## 核心功能亮点

### 1. 多数据库材料搜索
- **Materials Project**: 支持元素、化学式、带隙范围等条件搜索
- **OQMD数据库**: 支持元素组合、带隙范围、稳定性筛选
- **统一结果格式**: 返回材料ID、化学式、带隙、对称性等信息

### 2. 结构可视化
- **2D结构图**: 使用Matplotlib生成，通过文件服务器提供URL访问
- **3D交互结构**: 使用ASE生成HTML，支持旋转、缩放交互
- **自动嵌入**: AI对话中自动识别结构结果并嵌入可视化链接

### 3. VASP计算全流程管理
```
创建任务 → 上传CIF → 生成输入 → 修改INCAR → 提交计算 → 提取结果
```
- **统一远程连接**: 基于SSH/SFTP的远程服务器管理
- **任务目录结构**: 自动创建标准化目录结构
- **INCAR交互编辑**: 表格化参数编辑，支持实时保存
- **结果提取**: 支持relax（结构优化）、scf（自洽）、band（能带）、dos（态密度）结果解析

### 4. 机器学习集成
- **带隙预测**: 基于XGBoost的预训练模型
- **简单接口**: 只需输入化学式即可预测带隙值
- **可扩展**: 模块化设计支持添加更多ML模型

### 5. 项目管理与工作流
- **项目进度跟踪**: 记录材料研发各阶段状态
- **工作流定义**: 支持自定义研发流程步骤
- **JSON持久化**: 项目信息存储于`material_workflow.json`


### 环境变量配置
```bash
# DeepSeek API（必需）
DEEPSEEK_API_KEY=your_deepseek_api_key

# Materials Project API（材料搜索）
MP_API_KEY=your_mp_api_key

# SSH连接配置（VASP计算）
VASP_HOST=your_vasp_server_host
VASP_PORT=22
VASP_USERNAME=your_username
VASP_PASSWORD=your_password
VASP_BASE_DIR=/path/to/vasp/workspace
```

### 启动流程
1. **启动文件服务器**（自动由Agent初始化）
2. **初始化LangChain Agent**:
   ```python
   from agent.langchain_agent import create_langchain_agent
   agent = create_langchain_agent(api_key="your_deepseek_api_key")
   ```
3. **启动Web应用**:
   ```bash
   streamlit run web_app.py
   ```

## 使用示例

### 示例1：材料搜索与可视化
```python
# 搜索含Li、Co、O元素的材料，带隙0-2eV
result = agent.search_materials(
    elements=["Li", "Co", "O"],
    band_gap=(0, 2),
    chunk_size=10
)

# 获取特定材料的结构
structure = agent.get_material_structure(
    material_id="mp-12345",
    get_sites=True,
    get_plot=True
)
# 返回包含2D图片URL和3DHTML链接
```

### 示例2：VASP计算工作流
```python
# 1. 创建任务目录
agent.create_task("SiO2", "cifs/SiO2.cif")

# 2. 生成结构优化输入文件
agent.create_mission("SiO2_relax", "relax")

# 3. 修改INCAR参数
agent.modify_incar("SiO2_relax", "relax", read=False, write="ENCUT=520\nEDIFF=1E-5")

# 4. 提交计算
agent.submit_mission("SiO2_relax", "relax")

# 5. 提取结果
result = agent.extract_result("SiO2_relax", "relax", plot=True)
```

### 示例3：结构建模
```python
# 构建NaCl结构
structure = agent.build_structure(
    a=5.64, b=5.64, c=5.64,
    alpha=90, beta=90, gamma=90,
    elements=["Na", "Cl"],
    frac_coord=[[0,0,0], [0.5,0.5,0.5]],
    save_to_cif=True
)
```

## 技术挑战与解决方案

### 挑战1：远程计算管理
- **问题**: SSH连接不稳定，文件传输错误处理复杂
- **解决方案**: 实现重试机制，连接池管理，详细的错误日志

### 挑战2：结构可视化
- **问题**: 3D结构交互需要WebGL，2D结构需要图片生成
- **解决方案**: 文件服务器统一管理，生成可访问URL，支持多种格式

### 挑战3：工具调用编排
- **问题**: 复杂的工具依赖和调用顺序
- **解决方案**: LangGraph显式状态图，清晰的工具调用流程

### 挑战4：用户体验
- **问题**: 复杂的材料科学操作对非专家用户不友好
- **解决方案**: 自然语言接口，引导式Web界面，实时可视化反馈
