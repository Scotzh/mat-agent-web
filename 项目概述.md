# MatAgent 材料智能设计平台

## 项目概述

MatAgent 是一个面向材料科学领域的智能计算平台，基于 LangGraph 实现 Function Calling Agent，提供材料数据检索、晶体结构可视化、机器学习预测、VASP 计算任务管理等功能。

## 核心功能

### 🔍 材料查询
- 集成 Materials Project 和 OQMD 数据库
- 支持按元素、排除元素、化学系统、带隙范围等条件搜索
- 获取晶体结构、晶格参数、对称性等信息

### 📊 结构可视化
- 2D 结构图生成（使用 ASE 渲染）
- 3D 交互式 HTML 可视化
- CIF 文件下载

### 🧪 机器学习预测
- XGBoost 带隙预测模型
- 支持批量预测

### 💻 VASP 远程计算
- SSH 远程连接管理
- 任务创建、提交、结果提取
- INCAR 参数修改

### 💬 智能对话
- 基于 DeepSeek LLM 的自然语言交互
- 多轮对话支持
- 历史会话记录（SQLite 持久化）

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境变量

创建 `.env` 文件：

```bash
cp .env.example .env
```

必需配置：
- `mp_API_KEY` - Materials Project API Key
- `DEEPSEEK_API_KEY` - DeepSeek API Key
- SSH 连接信息（HOST, PORT, USERNAME, PASSWORD, base_dir）

### 3. 启动服务

```bash
# 启动 Flask 文件服务器
python flask_server.py

# 启动 Streamlit Web 界面
streamlit run web_app.py
```

### 4. 访问

浏览器打开 `http://localhost:8501`
